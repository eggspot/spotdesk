MainWindow Views/MainWindow.axaml

Primary application shell. Custom chromeless title bar with drag region. Three-zone layout: collapsible sidebar (240px), tab bar (40px height), and session pane. Sidebar collapses to icon-only (48px) via Ctrl+\. 

+-----------------------------------------------------------------------+
| TITLEBAR (32px, custom chrome)                                        |
| [=] EggDesk           [Search Ctrl+K]       [Sync ^] [Settings] [-X] |
+----------+------------------------------------------------------------+
|          | TAB BAR (40px)                                             |
| SIDEBAR  | [+ New] [* prod-web-01 x] [* db-server x] [...]           |
| (240px)  +------------------------------------------------------------+
|          |                                                            |
| Groups   |                  SESSION PANE                              |
| + My     |                                                            |
|   Servers|          (RDP / SSH / VNC renders here)                    |
|   > Prod |          Full resolution, no letterbox                    |
|   > Dev  |                                                            |
|   > Stg  |          When no session open:                            |
|          |             WelcomeView shown                              |
| Recents  |                                                            |
|  web-01  |          When session active:                              |
|  db-01   |             RdpView / SshView / VncView                   |
|  ci-box  |             via SessionViewSelector                       |
|          |                                                            |
| [+ Add]  |                                                            |
+----------+------------------------------------------[StatusBar]-------+
Keyboard shortcuts: Ctrl+K search, Ctrl+N new connection, Ctrl+\ toggle sidebar, Ctrl+1-9 switch tabs, Ctrl+W close tab. Sidebar resize via drag handle with pointer capture.

SidebarView Views/SidebarView.axaml

Connection tree with collapsible groups, inline search filter, and recent connections. Groups are recursive -- a group can contain child groups. Double-click group name to rename inline. Right-click for context menu. 

+--------------------------+
| [=]  EggDesk       [+]  |  <-- app name + new connection button
+--------------------------+
| * Search...       Ctrl+K |  <-- inline filter (separate from Ctrl+K overlay)
+--------------------------+
| v Favorites (3)          |  <-- pinned connections
|   * prod-web-01    [RDP] |
|   * db-server      [SSH] |
|   o backup-box     [VNC] |
+--------------------------+
| v Production (5)         |  <-- collapsible group
|   * web-01         [RDP] |
|   * web-02         [RDP] |
|   * db-primary     [SSH] |
|   o db-replica     [SSH] |
|   o cache-01       [SSH] |
+--------------------------+
| > Staging (3)            |  <-- collapsed group
+--------------------------+
| > Development (8)        |
+--------------------------+
|  ------- Recent -------- |
|   * db-primary     2m    |  <-- last connected time
|   o web-01         1h    |
+--------------------------+

Status dots:
  * green  = connected
  ~ amber  = connecting
  x red    = error
  o gray   = idle
Connection list items show: status dot + name + host (small) + protocol badge. Hover reveals connect button. Context menu: Connect | Edit | Duplicate | Delete | Pin.

SessionTab Controls/SessionTab.axaml

Individual tab in the tab bar. Shows protocol icon, hostname, connection status dot, and close button. Active tab has accent underline (2px). Middle-click to close. Tabs are capped at 200px width; overflow scrolls horizontally. 

+--- Tab Bar (40px height) -------------------------------------------+
|                                                                      |
| [+ New]  +----------------+  +----------------+  +----------------+ |
|          | [RDP] web-01 * x|  | [SSH] db-01 * x|  | [VNC] vnc-1 o x| |
|          +=====[accent]====+  +----------------+  +----------------+ |
|             ^active tab                                              |
+----------------------------------------------------------------------+

Single tab detail:
+---------------------------+
| [icon] hostname  [*] [x] |
+---------------------------+
  icon = protocol (RDP/SSH/VNC)
  [*]  = status dot (colored)
  [x]  = close button (11px)

  Active:  accent underline, lighter bg
  Hover:   BgHover background
  Default: transparent

RdpView Views/RdpView.axaml

Displays the remote desktop framebuffer as a WriteableBitmap inside an Avalonia Image. Auto-hide toolbar appears on mouse hover at top. Keyboard/mouse events forwarded to backend. Ctrl+Alt+Break releases keyboard focus. 

+----------------------------------------------------------------------+
| TOOLBAR (auto-hide on hover, 40px)                                   |
| [Fit Window] [Fullscreen] [Screenshot] [Ctrl+Alt+Del] [Disconnect]  |
+----------------------------------------------------------------------+
|                                                                      |
|                                                                      |
|            +--------------------------------------------+            |
|            |                                            |            |
|            |    Remote Desktop Framebuffer              |            |
|            |    (WriteableBitmap in Image control)      |            |
|            |                                            |            |
|            |    Resolution matches window size          |            |
|            |    All keyboard/mouse captured             |            |
|            |                                            |            |
|            +--------------------------------------------+            |
|                                                                      |
+----------------------------------------------------------------------+
| StatusBar: [* Connected] [prod-web-01] [RDP] [24ms] [1920x1080]     |
+----------------------------------------------------------------------+

Backend:
  Windows = WindowsRdpBackend (AxMSTscLib COM) -- STUB
  macOS   = FreeRdpBackend (P/Invoke)          -- PARTIAL
  Linux   = FreeRdpBackend (P/Invoke)          -- PARTIAL

SshView Views/SshView.axaml

Terminal emulator rendering into a WriteableBitmap. Target: JetBrains Mono 13px. VT100/xterm-256color support via Vt100Parser + TerminalBuffer. Currently placeholder -- renders dots instead of glyphs. 

+----------------------------------------------------------------------+
|                                                                      |
| JetBrains Mono 13px, background #0F1117                             |
|                                                                      |
| user@db-primary:~$ sudo systemctl status nginx                      |
| * nginx.service - A high performance web server                     |
|    Loaded: loaded (/lib/systemd/system/nginx...)                    |
|    Active: active (running) since Mon 2026-01...                    |
|                                                                      |
| user@db-primary:~$ _                                                |
|                                                                      |
|                                                                      |
|                                                                      |
+----------------------------------------------------------------------+
| [* Connected] [db-primary] [SSH] [12ms]                              |
+----------------------------------------------------------------------+

Terminal features (target):
  - VT100 + xterm-256color + truecolor SGR
  - Cursor movement, erase, scroll regions
  - 10,000 line scrollback buffer
  - 60fps render via DispatcherTimer

Current status: PLACEHOLDER
  - Vt100Parser: CSI sequences implemented
  - TerminalBuffer: char/attr grid works
  - SshView: renders single pixel per cell (no glyph rendering)

VncView Views/VncView.axaml

VNC session display. Same pattern as RdpView -- framebuffer in WriteableBitmap, auto-hide toolbar on hover. Backend: RemoteViewing library. Minimal implementation.

+----------------------------------------------------------------------+
|                                                                      |
|            +--------------------------------------------+            |
|            |                                            |            |
|            |    VNC Framebuffer                         |            |
|            |    (WriteableBitmap in Image control)      |            |
|            |                                            |            |
|            |    VncSessionManager connects via          |            |
|            |    RemoteViewing.VncClient                 |            |
|            |                                            |            |
|            +--------------------------------------------+            |
|                                                                      |
+----------------------------------------------------------------------+
| [* Connected] [vnc-server] [VNC] [5900]                              |
+----------------------------------------------------------------------+

Status: MINIMAL (~20%)
  - Connect/disconnect works
  - Framebuffer display works
  - Toolbar buttons NOT wired to commands
  - No input forwarding
  - Not integrated with main SessionManager
